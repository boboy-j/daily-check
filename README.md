# 📱 每日打卡 — 家庭习惯打卡应用

一个专为家庭设计的 **iOS 原生风格** 网页应用，支持添加到 iPhone 主屏幕，模拟原生 App 体验。适合家庭成员一起打卡、追踪习惯养成进度。

---

## ✨ 功能亮点

| 功能 | 说明 |
|------|------|
| 👨‍👩‍👧‍👦 **成员管理** | 添加/删除家庭成员，12 种表情头像自由选择 |
| 📋 **打卡任务** | 每日打卡，iOS Switch 风格勾选，一键切换日期回顾 |
| ⏱️ **倒计时任务** | 支持设置任务最低时长（分钟），倒计时结束有声音提醒 |
| 📊 **统计分析** | 成员完成率、家庭总览、任务完成率排行、30 天热力图 |
| 🔥 **连续打卡** | 自动计算每位成员的连续打卡天数 |
| 🎉 **完成动画** | 全部任务完成后触发礼花特效，仪式感满满 |
| 💾 **本地存储** | 所有数据保存在浏览器 localStorage，无需注册登录 |
| 📱 **PWA 支持** | 添加到 iPhone 主屏幕后全屏运行，无浏览器栏 |

## 🚀 如何使用

### 1️⃣ 直接打开

用浏览器打开 `index.html` 即可使用。

### 2️⃣ 添加到 iPhone 主屏幕（推荐）

1. 在 iPhone 上用 Safari 打开页面
2. 点击底部 **分享按钮** (⬆️)
3. 滑动找到并点击 **「添加到主屏幕」**
4. 点击右上角 **「添加」**
5. 主屏幕出现带图标的「每日打卡」，打开即全屏沉浸体验 ✅

> 💡 添加到主屏幕后，应用会全屏运行，无浏览器地址栏和工具栏，和原生 App 体验一致。

## 🎮 使用指南

### 👤 第一步：添加家庭成员

1. 在「成员」页点击右下角 **➕** 按钮
2. 选择头像（😊😎🤗😺🐶🐰🐼🦊🐸🌈🌟🌸）
3. 输入名称，点击「保存」

### 📝 第二步：创建打卡任务

1. 点击某个成员卡片进入打卡页面
2. 点击 **「管理任务」**
3. 输入任务名称（如：阅读、运动、练琴）
4. 如需定时：在「分钟」输入框填入时长（如：30）
5. 点击「添加」

### ✅ 第三步：每日打卡

1. 在打卡页面点击任务右侧的 **开关** 完成打卡
2. 有倒计时的任务会自动弹出计时器，点击「开始」
3. 倒计时结束自动完成，并有声音提示
4. 所有任务完成 → 礼花庆祝 🎉

### 📊 第四步：查看统计

- 底部导航栏切换到 **「统计」** 标签
- 可筛选 **全部成员 / 单个成员**
- 可切换 **近7天 / 近30天 / 近90天**
- 查看：完成率、连续打卡、任务排行榜、热力图

## 🎨 设计风格

iOS 原生设计语言，细致复刻 Apple 设计规范：

- **导航栏** — 44pt 高度，毛玻璃背景，0.5pt 分割线
- **Tab 栏** — 底部固定，毛玻璃效果，图标+标签
- **颜色** — iOS 蓝 `#007AFF` / 绿 `#34C759` / 红 `#FF3B30`
- **字体** — San Francisco / PingFang SC 系统字体栈
- **控件** — iOS Switch 开关、Action Sheet 弹窗、底部模态页
- **动画** — 原生级过渡曲线 `cubic-bezier(0.32, 0.08, 0.24, 1)`
